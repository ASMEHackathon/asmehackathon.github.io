Description

This dataset includes the files pertaining to a 3D additive manufacturing build performed on the Additive Manufacturing Metrology Testbed (AMMT) by Ho Yeung on July 8, 2018. The files include the input command files and in-situ process monitoring data, and metadata. This data is the first of future planned AMMT Process Monitoring Reference Datasets, as part of the Metrology for Real-Time Monitoring of Additive Manufacturing project.
